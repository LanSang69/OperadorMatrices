# Operador de matrices

## Descripción
Operador de matrices con la capacidad de llevar a cabo
cinco operaciones fundamentales:
• Sumatoria.
• Resta.
• Multiplicación entre matrices.
• Multiplicación de matriz por un vector.
• Suma de las columnas de una matriz.

## Requisitos Previos
Asegúrate de tener lo siguiente instalado en tu sistema:
- Node.js: Puedes descargarlo desde [https://nodejs.org/](https://nodejs.org/).
- TypeScript: Puedes instalarlo globalmente con `npm install -g typescript`.

## Configuración
1. Clona este repositorio: `https://github.com/LanSang69/OperadorMatrices.git`
2. Navega al directorio del proyecto: `OperadorMatrices/`

## Uso
1. Abre el archivo `index.html` en tu navegador web.
2. Llena los campos correspondientes a el número de filas y columnas
3. Presiona 'OK'
4. Ingresa los valores de las matrices y preciona 'operar'
